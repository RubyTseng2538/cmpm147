Experiment 04: Infinite Worlds

Description

In this experiment, I created an infinite flower grove with various elements of a flower grove.

Technical

I use % for randomness and increase/decrease the number behind % based on how often I want the tile to spawn.

Reflection

I find myself having issues with both aligning the tiles to look nice and make the selected tile and flower spawn be accurate. It is accurate when the user didn't move and click on the tiles at the start, but if the user move out into the infinite world, the flower doesn't spawn at the time user clicked.
